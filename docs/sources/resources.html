Resources for learning R for undergrads

Below are a list of resources which I have found useful. Please feel free to ask me about any of these or share ones you know about.

Data Science

Modern Dive

  • This book has both basic data analysis and statistics. Its got nice examples but there aren’t many homework problems.

Modern Data Science with R

  • This book is a complete data science and statistics text with R. Many good problems, especially if you like baseball.

R for Data Science

  • I find this to be a great reference manual.

Statistics

Project Mosaic

  • The mosaic package was made by faculty to offer a bridge into R via statistics. I have taught with it many times in introductory statistics.

  • Statistical Modeling: A Fresh Approach by Daniel Kaplan

    • I have not used this text, but it has a booklet on R to acompany it.

  • A Student’s Guide to R

    • This is a helpful resource to learning what functions in the mosaic package do.

  • R tutorials

    • I made some learnr tutorials to show how to use the mosaic package with for Intro stats. The examples come from OpenIntroStats 4th edition. They are beta, if you want to collaborate on them let me know.

Introduction to Modern Statistics

  • This book uses infer which is a more professional tool than mosaic.

  • I am currently teaching with this text. website

Programming

Hands on programming with R

  • This is the next text I plan to read through. Its been recommended several times.
